ELEMENTI DI TESTO

Elemento STRONG parte di default con il grassetto

Elemento U(underline) parte di default sottotitolato

Elemento SPAN non ha stili di default, e' quello piu generico, e' quindi il piu' manipolabile

Puoi assegnare classi a ognuno di questi elementi

Esempio span: Ciao, sono un link

Invece se vuoi aggiungere spazi nel testo: TESTO LONTANO
